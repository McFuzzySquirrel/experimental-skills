# Example Output Guidance

Load this reference only when the user asks for a sample output artifact, formatting example,
or quality bar before generating meeting minutes.

## What a good output demonstrates

- Clear TL;DR with the main outcome and next step.
- A chronological summary that synthesizes discussion rather than copying transcript lines.
- Decisions with explicit rationale.
- Action items with owners (or `TBD`) and priorities.
- Open questions limited to unresolved items; if none, say "None identified."

## Mini example shape

Use this as a quality cue, not a strict template replacement:

- TL;DR: one short paragraph (3-5 sentences).
- Summary: concise narrative (200-400 words).
- Decisions table: each row maps to a concrete commitment.
- Action Items table: verb-led tasks with owner and priority.
- Open Questions table: only unresolved points.

Do not include transcript-style back-and-forth, speculative interpretation, or invented details.
